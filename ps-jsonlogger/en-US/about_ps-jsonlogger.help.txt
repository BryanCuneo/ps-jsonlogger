TOPIC
    about_ps-jsonlogger

SYNOPSIS
    Lightweight, dependency-free JSON logger for PowerShell scripts.

LONG DESCRIPTION
    ps-jsonlogger is a small, dependency-free structured logging module for
    PowerShell that offers both compact JSON logs on-disk and human-readble
    console output. It supports log levels, context objects, full call stack
    inclusion, and more."

EXAMPLES
    Import-Module ps-jsonlogger

    $logger = New-JsonLogger `
        -LogFilePath "./basic_logging_example.log" `
        -ProgramName "Basic Logging Example" `
        -WriteToHost

    $logger.Log("Hello, World!")
    $logger.Log("INFO", "INFO is the default, but you can also specify it.")
    $logger.Log("WARNING", "This text will be yellow in the console.")
    $logger.Log("ERROR", "And this text will be red.")
    $logger.Log("VERBOSE", "VERBOSE logs always include the full call stack.")

    $logger.Close("You can write a final log entry to the file like this.")
    $logger.Log("FATAL", "For terminating errors, use FATAL to call 'exit 1'.")
    $logger.Log("DEBUG", "This line will never be logged.")
